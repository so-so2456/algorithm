이진 탐색을 이해하기 전에, 순차 탐색을 먼저 이해해야 한다

===순차 탐색===
리스트 안에 있는 특정한 데이터를 찾기 위해
앞에서부터 데이터를 하나씩 차례대로 확인하는 방법

순차 탐색은 자주 사용되는 알고리즘이다

===이진 탐색===
리스트 내에서 데이터를 매우 빠르게 탐색하는 알고리즘이다

이진 탐색은 배열 내부 데이터가 정렬되어 있어야 한다

탐색 범위를 절반씩 좁혀가며 데이터를 탐색하는 특징이 있다

위치를 나타내는 변수 3개(시작점, 끝점, 중간점)를 사용한다

찾으려는 데이터와 중간점 위치에 있는 데이터를 반복적으로 비교해서
원하는 데이터를 찾는다
