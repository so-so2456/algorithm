====구현 알고리즘====

언어 문법에 능숙하면서 코드 작성 속도가 빠른 것을 요구하는 문제

> 종류
 - 완전 탐색(모든 경우의 수를 전부 계산)
 - 시뮬레이션(문제에서 제시한 알고리즘을 한 단계씩 차례대로 수행)

> 유의 사항
 - C/C++ >> 자료형 크기 유의
    int                 > 2,147,438,647
    long long           > 9,223,372,036,854,775,807
    BigInteger(class)   > 제한 없음

 - Python >> 리스트 크기 유의
    리스트 길이      메모리 사용량
    1,000           약 4KB
    1,000,000       약 4MB
    10,000,000      약 40MB
